HTML (Hypertext Markup Language)

HTML itu adalah kerangka dasar dari suatu WebSite yang ingin kita buat. Nah kalau bisa kita ibaratkan, jadi HTML ini itu kaya pondasi rumah, dinding rumah, sama atap rumahnya. Jadi HTML ini yang menjadi struktur dasar rumah kita. Jadi, kita bisa bayangkan apa bila gak ada HTML, maka website kita itu cuma kaya teks polos gak jelas. Gak ada

Jadi ada beberapa fungsi utama dari HTML ini seperti :

  • Membuat struktur dasar halaman web.
  • Menyusun konten dengan elemen-elemen yang terstruktur.
  • Memasukkan gambar, video, dan file multimedia.
  • Menciptakan tautan (hyperlink) ke halaman lain.
  • Menangani formulir untuk input pengguna.

CSS (Cascading Style Sheets)

CSS adalah bahasa style sheet yang digunakan untuk mengatur tampilan visual halaman web yang dibuat dengan HTML. Dengan CSS, kita dapat mengubah _style_ elemen-elemen HTML, seperti warna, font, tata letak, dan responsivitas.

Fungsi Utama CSS

  • Mengontrol tampilan (style) visual elemen HTML.
  • Mengatur tata letak (layout) halaman web.
  • Membuat halaman web responsif di berbagai perangkat.
  • Memisahkan style dari konten HTML untuk pengelolaan lebih baik.
  • Mengatur animasi dan efek visual.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat interaksi dan dinamika pada halaman web. Dengan JavaScript, kita dapat membuat efek animasi, validasi form, mengubah konten halaman secara dinamis, dan berkomunikasi dengan server.

Fungsi Utama JavaScript

  • Membuat interaksi pengguna (misalnya: tombol, form).
  • Mengubah konten halaman secara dinamis (misalnya: memperbarui data).
  • Melakukan validasi form (misalnya: memastikan input valid).
  • Membuat efek animasi dan transisi.
  • Mengelola data dan berinteraksi dengan API.
  • Berinteraksi dengan server (melalui AJAX).

Kesimpulan

HTML, CSS, dan JavaScript adalah tiga teknologi inti yang bekerja sama untuk menciptakan halaman web modern. HTML menyediakan struktur, CSS menyediakan _styling_, dan JavaScript menyediakan interaksi. Dengan menguasai ketiga teknologi ini, pengembang dapat membangun website yang fungsional, menarik, dan interaktif.
